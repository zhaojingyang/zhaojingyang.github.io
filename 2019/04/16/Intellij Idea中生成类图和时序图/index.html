Intellij Idea中生成类图和时序图

发表于 2019-04-16 | 分类于 Tools | | 阅读次数: |
字数统计: 305 字 | 阅读时长 ≈ 1 分钟

类图

Idea中有根据代码生成类图的功能，mac下快捷键是shift+option+command+U,笔者给改成了control+option+command+U,弹出pop页面是option+command+U。可以通过设置->keymap-搜uml查看和设置快捷键。


当在子类中按下快捷建时候会自动搜到父类加入到图中，如下图显示spring的BeanWrapperImpl相关类图。工具栏上可以选择设置类图中显示的属性方法等。

如果想加入其他类，可以按[回车]快捷键，输入类名查找到指定的类加入到图中


可以在类图上邮件点击jump到相关资源文件等操作。

时序图SequenceDiagram

需要安装SequenceDiagram插件,地址SequenceDiagram安装如下图

安装成功后可按上步中keymap中查询sequence并设置快捷键，笔者设置的是control+option+command+S,或在Tools中点击此功能，如下图显示spring getBean的时序图及对应的代码。(注意对于spring这种源码很复杂的项目生成时序图时call depth不要设置太大否则idea会oom)
